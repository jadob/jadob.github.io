What is Jadob?

Jadob is a PHP7 application framework, based on Symfony and Zend components. It comes with the most important stuff needed to start developing your PHP project. Jadob uses Twig for templating and Doctrine DBAL to work with databases. There is no Doctrine ORM Support available, but you can easily add it using Service providers.
